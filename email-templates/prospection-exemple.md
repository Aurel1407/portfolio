# Exemple d'utilisation du template prospection.html

## Scénario : Prospection d'un restaurant local sans site web

### Informations collectées

- **Entreprise** : Restaurant Le Jardin Gourmand
- **Contact** : Monsieur Dupont (gérant)
- **Activité** : Restaurant traditionnel français
- **Localisation** : Cholet
- **Constat** : Pas de site web, seulement une page Facebook peu active

### Variables à remplacer

```
{{company_name}}      → Restaurant Le Jardin Gourmand
{{contact_name}}      → Monsieur Dupont
{{business_activity}} → Un site bien référencé pourrait vous permettre d'apparaître lorsque quelqu'un recherche "restaurant Cholet" ou "cuisine du terroir". Cela faciliterait aussi les réservations en ligne.
```

**Pourquoi cette formulation est puissante :**

- Projette mentalement le prospect dans le résultat concret
- Mentionne des requêtes Google réelles qu'il comprend
- Ajoute un bénéfice opérationnel (réservations)

### Objet d'email suggéré

```
Restaurant Le Jardin Gourmand : vos clients vous cherchent déjà sur Google
```

### Résultat attendu

Un email personnalisé qui :

- Mentionne précisément le restaurant
- Montre que tu connais leur activité
- Souligne le manque de présence web sans être insistant
- Présente les bénéfices concrets (réservations en ligne, visibilité Google, menu accessible 24/7)
- Propose un échange simple (café, appel, email)

### Timing de suivi

1. **J+0** : Envoi de l'email le mardi ou mercredi matin (9h-11h)
2. **J+2** : Appel téléphonique courtois si pas de réponse
   - "Bonjour, Aurélien Thébault. Je vous ai envoyé un email concernant votre présence en ligne. Avez-vous eu le temps d'y jeter un œil ?"
3. **J+7** : Email de relance léger si toujours pas de retour
4. **J+30** : Clôture du dossier (ne pas harceler)

### Taux de conversion attendu

- **Taux d'ouverture** : 30-40% (prospection B2B locale)
- **Taux de réponse** : 5-10% (avec personnalisation forte)
- **Conversion en RDV** : 2-5% (si bon ciblage)

Pour 20 emails envoyés :

- 6-8 ouvertures
- 1-2 réponses
- 0-1 rendez-vous qualifié

### Astuces de personnalisation avancée

Pour augmenter les chances de réponse :

1. **Remplir systématiquement {{business_activity}}** (CRITIQUE)
   - "J'ai vu que vous proposez de la plomberie et du chauffage. Un site web permettrait de mettre en avant vos interventions d'urgence et faciliter les demandes de devis."
   - "Vous êtes spécialisé en menuiserie sur mesure. Un site pourrait présenter vos créations et rassurer les clients sur votre savoir-faire."
   - "Votre restaurant propose une cuisine du terroir. Un site permettrait de présenter vos menus, horaires et faciliter les réservations."

   ⚠️ **Cette phrase prouve que l'email n'est pas générique = taux de réponse x3**

2. **Adapter le ton selon le secteur**
   - Restaurant gastronomique → Ton élégant, mentions "identité", "expérience"
   - Artisan local → Ton direct, mentions "proximité", "savoir-faire"
   - Commerce → Ton dynamique, mentions "clients", "ventes"

3. **Proposer une valeur immédiate en bonus**
   - "Je peux vous créer une page Google Business gratuite si vous n'en avez pas"
   - "Je peux analyser gratuitement votre visibilité actuelle sur Google"

### Ce qu'il NE FAUT PAS faire

❌ Copier-coller le même mail à 50 entreprises
❌ Envoyer le vendredi après-midi ou le lundi matin
❌ Utiliser des objets "clickbait" ("Urgent", "Offre limitée")
❌ Relancer plus de 2 fois
❌ Mentir sur les statistiques ou les tarifs
❌ Critiquer leur absence de site de manière négative

### Exemples de {{business_activity}} selon le secteur

**🔥 Version "projection mentale" (ULTRA PERFORMANTE)**

**Restaurant :**

```
Un site bien référencé pourrait vous permettre d'apparaître lorsque quelqu'un recherche "restaurant Cholet" ou "cuisine du terroir". Cela faciliterait aussi les réservations en ligne.
```

**Plombier :**

```
Un site optimisé pourrait vous faire apparaître quand quelqu'un tape "plombier urgence Cholet" ou "dépannage chauffage". Vos clients pourraient aussi demander des devis 24/7.
```

**Commerce :**

```
Un site bien référencé vous rendrait visible quand on recherche "boutique artisanale Cholet" ou "produits du terroir". Vous pourriez aussi y afficher vos horaires et votre catalogue.
```

**Architecte :**

```
Un site professionnel vous positionnerait sur des requêtes comme "architecte Cholet" ou "rénovation maison". Cela renforcerait aussi votre crédibilité avec un portfolio en ligne.
```
