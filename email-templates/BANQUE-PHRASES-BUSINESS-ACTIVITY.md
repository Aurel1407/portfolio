## 🚫 Ce qu'il NE FAUT PAS écrire

### ❌ Phrases génériques (taux réponse 2-4%)

```
❌ "Un site web vous serait utile."
❌ "Vous devriez avoir un site internet."
❌ "Je peux vous créer un site web professionnel."
❌ "Avoir un site est important de nos jours."
```

### ❌ Phrases techniques (perdent le prospect)

```
❌ "Je peux vous créer un site avec React et Node.js."
❌ "Votre SEO pourrait être amélioré avec un site optimisé."
❌ "Un CMS WordPress avec WooCommerce serait idéal."
```

### ❌ Phrases trop commerciales (spam)

```
❌ "OFFRE SPÉCIALE : Site web à 500€ seulement !"
❌ "Dernière chance d'avoir un site professionnel !"
❌ "Ne passez pas à côté de cette opportunité !"
```

Avant d'envoyer, assure-toi que ta phrase {{business_activity}} :

✅ Ajoute 1-2 bénéfices concrets opérationnels

## 💡 Astuce pro : Vérifier les requêtes Google

Avant d'envoyer, tape la requête sur Google pour vérifier :

1. **Volume de recherche** : Des résultats apparaissent ? ✅
2. **Concurrence** : Le prospect n'apparaît pas ? ✅
3. **Pertinence** : La requête correspond à l'activité ? ✅

**Exemple :**

```
Recherche Google : "plombier urgence Cholet"
→ 12 000 résultats
→ Le prospect n'apparaît pas dans les 3 premières pages
→ ✅ Parfait pour la prospection
```
