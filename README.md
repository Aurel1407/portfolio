# Portfolio - Aurélien Thébault

Site vitrine orienté acquisition clients pour artisans du bâtiment. Aurélien Thébault (Auvielno) crée des sites qui génèrent des demandes de devis — plombiers, électriciens, maçons, couvreurs, menuisiers et tous les métiers du bâtiment.

**[Voir le site en ligne](https://auvielno.fr/)**

## Stack Technique

- **Frontend** : HTML5, Tailwind CSS, JavaScript (Vanilla)
- **Outils** : Git, GitHub Pages (domaine personnalisé), Font Awesome, Google Fonts
- **Accessibilité** : ARIA labels, navigation clavier, contraste optimisé
- **Performance** : Images WebP, lazy loading, animations GPU

## Fonctionnalités

- Interface responsive (mobile-first)
- CTAs collants conversion (WhatsApp + contact)
- Animations CSS personnalisées
- Protection anti-spam des coordonnées (JavaScript obfuscation)
- Formulaire de contact intégré (Formspree)
- Navigation smooth avec indicateur de section active
- Accessibilité renforcée (WCAG 2.1 AA)
- SEO optimisé (Schema.org, Open Graph, meta tags)

## Structure

```
.
├── index.html
├── projects/           # Pages études de cas clients

```

## Installation Locale

```bash
# Cloner le repository
git clone https://github.com/Aurel1407/new_portfolio.git

# Ouvrir index.html dans un navigateur
# Ou utiliser un serveur local (Live Server, http-server, etc.)

```

## Déploiement

Le site est déployé sur **GitHub Pages** avec domaine personnalisé **auvielno.fr**.

Toute modification poussée sur `main` déclenche un redéploiement automatique.

## Projets Présentés

- **ZetouPanda** : Site e-commerce artisan (React, TypeScript, Node.js, SQL)
- **Shu-no** : Plateforme de réservation (React, TypeScript, Node.js, Express)
- **ClimatServ17** : Site vitrine artisan (Next.js, TypeScript, Tailwind CSS, React)
- **APE La Croixille** : Site associatif (React, TypeScript, Node.js, Express, PostgreSQL)
- **Calcul Trigo** : Outil web éducatif (HTML5, CSS3, JavaScript Vanilla)
- **Portfolio** : Site personnel (HTML5, Tailwind CSS, JavaScript)

## Optimisations Sécurité

- Content Security Policy (CSP) stricte
- Protection anti-spam des contacts (obfuscation JavaScript)
- Validation des formulaires côté client
