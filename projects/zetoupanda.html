ZetouPanda

Application E-commerce Full-Stack

En développement 2025

À propos du projet

ZetouPanda est une application e-commerce complète développée dans le cadre de ma formation de développeur web full-stack. Ce projet personnel ambitieux combine une interface utilisateur moderne avec un système de gestion complet côté administrateur.

L'application permet aux utilisateurs de parcourir un catalogue de produits, de gérer leur panier et de passer des commandes, tandis que l'interface d'administration offre une gestion complète des produits, des commandes et des utilisateurs.

Technologies utilisées

Frontend

React.js TypeScript Tailwind CSS React Router Context API

Backend

Node.js Express MySQL JWT Bcrypt

Fonctionnalités principales

Gestion du panier

Ajout, modification et suppression de produits avec calcul automatique du total

Authentification

Système d'inscription et connexion sécurisé avec JWT

Catalogue produits

Navigation par catégories avec recherche et filtres avancés

Gestion des commandes

Suivi des commandes de la création à la livraison

Panel Admin

Interface d'administration complète pour gérer le site

Responsive Design

Interface adaptée à tous les appareils (mobile, tablette, desktop)
