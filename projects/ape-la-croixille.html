APE La Croixille

Site associatif full-stack avec espace administrateur, actualités et gestion d'événements.

Projet réalisé 2026 WCAG 2.1 AA

À propos du projet

APE La Croixille est une plateforme web développée pour l'Association des Parents d'Élèves de l'école de La Croixille. L'objectif est de centraliser les informations importantes pour les familles : événements, actualités et communication.

Le projet inclut une partie publique claire et mobile-first, ainsi qu'un espace administrateur sécurisé permettant la gestion complète des contenus (articles, événements, catégories) avec une attention particulière portée à l'accessibilité. Le site a été développé gratuitement pour soutenir l'association locale.

Technologies utilisées

Frontend

React 18 TypeScript Vite React Router Tailwind CSS Zustand

Backend

Node.js Express.js PostgreSQL Sequelize JWT REST API

Fonctionnalités principales

Actualités & articles

Publication et organisation des contenus avec catégories dédiées.

Gestion d'événements

Création, modification et affichage des événements à venir.

Espace admin sécurisé

Authentification JWT et gestion des droits d'administration.

Responsive mobile-first

Interface optimisée pour smartphones, tablettes et desktop.

Accessibilité AA

Navigation clavier, contraste conforme et support lecteurs d'écran.

Architecture full-stack

Frontend et API découplés pour une maintenance et évolution facilitées.
